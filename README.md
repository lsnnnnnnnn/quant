# A 股日频横截面多因子研究入门

## 项目目标

本仓库是一套面向量化研究新人的 A4 中文 LaTeX 技术讲义，围绕“宽基股票池三因子周频约束多头策略”，串联从研究假设、点时数据、因子检验到含成本组合回测的完整基础流程。

项目首先追求时序正确、可核对、可复现和可教学，不以回测收益最大化为目标，也不把示例或模板表述为真实实证结果。

## 讲义内容

主体固定为十章：

1. 研究问题与可证伪假设；
2. 数据与时间边界；
3. 股票池、可交易边界与收益标签；
4. 因子构造与实现核对；
5. 因子预处理与基础暴露控制；
6. 单因子检验与可交易性诊断；
7. 稳健性、样本外与反过拟合；
8. 因子冗余与简单多因子合成；
9. 从综合得分到约束型多头组合；
10. 含成本回测与结果解释。

配套附录包括：

## 编译

根目录的 `main.tex` 是唯一正式编译入口，文档使用 `ctexbook`、A4 版式和 XeLaTeX。`.latexmkrc` 已设置默认入口与编译器。

推荐在仓库根目录执行：

```bash
latexmk -xelatex -interaction=nonstopmode -halt-on-error main.tex
```

若环境没有 `latexmk`，至少连续运行两次：

```bash
xelatex -interaction=nonstopmode -halt-on-error main.tex
xelatex -interaction=nonstopmode -halt-on-error main.tex
```

在 Overleaf 中，将 Main document 设为根目录的 `main.tex`，并选择 XeLaTeX 编译器。仓库默认不版本化 PDF 及 `.aux`、`.log`、`.toc`、`.out`、`.fls`、`.fdb_latexmk`、`.synctex` 等构建产物。

## 研究边界

- 当前仓库没有真实金融数据，也没有真实数据回测产物；
- 匿名案例、极小玩具数字、空白表格和结构图仅用于教学或核对，不代表真实证券或策略表现；
- 交易制度、数据字段、费用、价格代理、资金规模和组织流程等具体口径均需结合实际环境确认；
