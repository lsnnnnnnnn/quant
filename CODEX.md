# Codex 项目入口

本仓库的 Codex 自动工作指令位于根目录：

```text
AGENTS.md
```

请以 `AGENTS.md` 为唯一仓库级指令来源。本文件仅作为人工入口，避免出现两套重复且可能漂移的规则。

首次任务建议：

```text
请先读取根目录 AGENTS.md，并对仓库进行只读状态检查。不要修改任何文件。请报告：
1. 当前分支、最近提交和未提交修改；
2. 现有 LaTeX/Notebook/配置/AI 日志结构；
3. main.tex 是否是 Overleaf 可用主入口；
4. XeLaTeX 或 latexmk 是否能完成编译；
5. 对照 P03，哪些内容已具备、哪些缺失、哪些存在冲突；
6. 建议的最小下一步。
```
